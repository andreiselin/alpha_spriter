Коралловый клуб
Путеводитель


есть две картинки:
image_rgb
image_alpha

и есть json-объект с описанием:
{
    width:800px,
    height:500px,
    rgb: jpg,  // gif / #f5f5f5
    alpha:gif, // none / svg?
    elements:[
        {id,x,y,w,h,},
        {id,x,y,w,h,},
        {id,x,y,w,h,}
    ]
}

скрипт загружает эти две картинки (если они уже не загружены),
создает процедурные объекты канвас
из двух картинок генерится rgba-изображение
изображение режется на rgba-спрайты
созданные спрайты назначаются имеющимся <img> тегам по id



--------------



Зачем все это:
Позволяет маскировать полноцветные изображения, фотографии.
Ускорить загрузку изображений засчет собирания в сет спрайтов, оптимизации пустого места.
Ускорить загрузку засчет индексированного альфа-канала.
